Managing Your Games

Adding a New Game

  • 01 Open your Dashboard and locate the action bar.
  • 02 Click Add New Game to create a dedicated directory.
  • 03 Define the title and add optional notes for easy organization.

Uploading Saves

  • 01 Enter the game profile from your main drive list.
  • 02 Select your save file or a .zip archive of the save folder.
  • 03 Label the backup (e.g., "Post-Boss Save") and finalize the upload.

Data Integrity & Restoration

How to Restore a Backup

Restoring is as simple as downloading. Your files are stored in IndexedDB, making them persistent within your browser's local environment.

Find the save entry in your game's history.
Click the download icon to retrieve your file.
Place the file back into your local game directory.
Security Alert

While files are redundant, clearing your "Site Data" or "Browsing History" for this domain may wipe your IndexDB. To prevent this, install GameDrive as a PWA (App) to isolate your storage.

Common Questions

Folder Uploads

If your game uses a directory instead of a single file, compress it into a .zip before uploading. GameDrive treats archives as single units for efficiency.

Cross-Device Sync

Since this version uses local storage, you can use the Settings > Export tool to move your entire drive to a new computer in seconds.

PWA Benefits

Installing as an app (Chrome: "Install", Safari: "Add to Home Screen") prevents browser auto-cleanup from deleting your save history during disk maintenance.

Delete Safety

Deleting a save from GameDrive is permanent. We recommend keeping "Old" saves rather than deleting them, as cloud storage is highly optimized.
